About Me

Nama saya Muhammad Gilang Nur Afandi, biasa dipanggil Gilang. Saya berumur 16 tahun, dan saya pelajar di SMK N 2 Karanganyar, saya kelas XI RPL B. RPL (Rekayasa Perangkat Lunak) adalah salat satu jurusan yang saya minati di SMK N 2 Karanganyar. Saya mudah untuk beradaptasi, dapat bekerja keras, bekerja sama dalam tim, mudah berkomunikasi dengan orang lain serta senang mempelajari hal baru.

Saya adalah anak kedua dari 3 bersaudara. Hobi saya adalah menggambar, membaca, dan menonton film. Saya suka menggambar, walaupun saya tidak terlalu mahir. Saya menggambar hanya kadang-kadang. Saya juga suka membaca, saya pembaca yang random karena saya suuka membaca komik, cerita novel, ensiklopedia, dll. Saya suka film yang bergenre Action, Horror,Crime, Thriller, Comedy, serta Mystery.
